# 任務

完成任務獲得獎勵

## 任務介紹

貓芽內最主要獲得貓幣和積分點的方法  
任務每日00:00(UTC+8)刷新  
一般玩家每日2個任務，月卡玩家每日3個任務，可能刷到重複的任務
